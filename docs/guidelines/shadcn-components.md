The following official shadcn components are available. Check whether they are already installed under `apps/frontend/src/components/ui` before installing.

The `shadcn-ui` package is deprecated. Use `shadcn` instead:

```bash
yarn shadcn add <component-id> --cwd apps/frontend
```

## New components (chat / conversation)

| # | Component ID (slug) | Human-friendly name | What it's for |
| --- | --- | --- | --- |
| 1 | attachment | Attachment | File or image attachment with media, metadata, upload state, and actions. |
| 2 | bubble | Bubble | Conversational content in a message bubble; variants, alignment, grouping, reactions, collapsible content. |
| 3 | marker | Marker | Inline status, system note, bordered row, or labeled separator in a conversation. |
| 4 | message | Message | Message in a conversation with optional avatar, header, footer, and alignment. |
| 5 | message-scroller | Message Scroller | Chat scroll container that anchors turns, follows streamed responses, loads history, and jumps to messages. |
